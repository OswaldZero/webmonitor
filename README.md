# 基于调用winpcap接口而实现的网络流量分析系统
> 该系统是在我们计网老师的要求下设计的一款网络流量分析系统，
> 目前功能比较简单，只能支持一些简单协议的分析与解析（http,tcp,udp,ipv4,ipv6,arp,icmp,Ethernet）这些协议的解析，
